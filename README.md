Backend of the Multi Quest

Laravel + Filament

run backend:
```bash
  php artisan serve
```
run npm:
```bash
  npm run dev/prod
```
run reverb:
```bash
  php artisan reverb:start
```
listen to notifications:
```bash
  php artisan queue:listen
```

Setup:
```bash
    composer install
    
    php artisan migrate
    
    php artisan db:seed
    
    php artisan shield:setup
